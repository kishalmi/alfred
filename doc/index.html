command line arguments

alfred [-p PORT] [ROOT]

e.g. alfred -p 80 ~/htdocs

  • -p PORT

    TCP port to listen for connection

  • -i INJECT

    JS file to inject into all served html docs

  • ROOT

    directory to serve from

  • -s

    static mode - do not monitor changes
